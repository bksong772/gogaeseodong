> 순서대로 진행. 완료하면 [ ] → [x]. 막히면 그냥 두고 다음으로.

- [x] 그림의방 소개글 교체 (카카오톡/계좌이체 문구 삭제 포함)
- [ ] 취향의방 제목 + 소개글 교체
- [ ] 필사의방 제목 + 소개글 교체 (카카오톡 문구 삭제 포함)
- [ ] 대표 사진 순서 재배열 (그림의방: 옥탑방 사진 1번으로)

### 직예약 시스템 🧠
> Tally.so 없이 카카오 채널 + 계좌이체로 운영

- [ ] 카카오 채널 예약 접수 자동응답 문구 세팅 (AI에게 요청)
- [ ] 카카오 채널에 공간별 직예약 안내 버튼 추가
- [ ] 직예약 5% 할인 안내 문구 등록

### 카카오 채널 FAQ 챗봇 🧠 (비용 0원 / 30분)
> 카카오 채널 관리자 → 자동응답 → 키워드 추가

- [ ] 키워드 자동응답 세팅 (AI에게 "카카오 자동응답 문구 써줘" 요청)
  - 주차 / 와이파이 / 체크인 시간 / 체크아웃 시간
  - 예약 / 취소 / 환불 / 인원 추가
  - 가이드북 / 맛집 / 주변 관광

### 구글 캘린더 자동 연동 🧠 (비용 0원 / 10분)
> 에어비앤비 + 스페이스클라우드 예약이 구글 캘린더에 자동 표시

- [ ] 에어비앤비 → iCal URL 복사 (에어비앤비 앱 → 달력 → 달력 내보내기)
- [ ] 구글 캘린더 → URL로 캘린더 추가
- [ ] 스페이스클라우드 iCal URL도 동일하게 추가

### 예약 확정 자동 메시지 🧠 (비용 0원 / 15분)
> 에어비앤비 예약 확정 시 카카오 채널 안내 자동 발송

- [ ] 에어비앤비 메시지 템플릿 작성 (AI에게 요청)
- [ ] 예약 확정 트리거로 자동 발송 설정

## PHASE 4. 서동영화·서동부엌 매출 2배 전략 🧠
> 목표: 이번 달 서동영화·서동부엌 예약 2배↑
> 핵심 문제: 네이버에서 "서동영화" "서동부엌" 단독 검색 시 노출 안 됨
> → "오티티룸 서동영화" "공유주방 서동부엌" 으로 검색해야 나옴

**네이버 검색 노출 개선 (브랜드명 단독 검색)**
- [ ] 네이버 블로그 포스팅: "서동영화" 키워드 포스팅 3편+ (AI에게 초안 요청)
- [ ] 네이버 블로그 포스팅: "서동부엌" 키워드 포스팅 3편+ (AI에게 초안 요청)
- [ ] 구글맵 "서동영화" "서동부엌" 단독 등록 확인 (없으면 신규 등록)
- [ ] 인스타 해시태그 통일: #서동영화 #서동부엌 + #오티티룸서동영화 #공유주방서동부엌 병행 사용

**채널 소개글 리뉴얼**
- [ ] 서동영화 네이버 플레이스 소개글 리뉴얼 (후킹 문구 + 용도 다양화)
- [ ] 서동영화 스페이스클라우드 소개글 리뉴얼 (클래스룸·파티룸·소셜링 키워드 추가)
- [ ] 서동부엌 네이버 플레이스 소개글 리뉴얼 (후킹 문구 + 용도 다양화)
- [ ] 서동부엌 스페이스클라우드 소개글 리뉴얼 (쿠킹클래스·파티·소셜링 키워드 추가)

**예약 2배 만들 신규 타겟**
- [ ] 서동영화 → 소규모 영화 모임·팀 회식·생일파티 타겟 홍보 (AI에게 카피 요청)
- [ ] 서동부엌 → 쿠킹클래스·원데이클래스·소셜링 타겟 홍보 (AI에게 카피 요청)
- [ ] 평일 할인 패키지 안내 카카오 채널 등록

## AI에게 요청 가능한 것

```
"에어비앤비 카피 써줘"      → 각 방 리스팅 제목 + 설명
"릴스 캡션 써줘"            → 인스타 캡션 5개
"블로그 포스팅 초안"        → SEO 최적화 포스팅
"카카오 자동응답 문구 써줘" → 직예약 안내 메시지
"가이드북 내용 써줘"        → 서동 맛집/코스/역사 HTML
"매뉴얼 내용 써줘"          → FAQ, 체크인 안내
"체크리스트 현황 봐줘"      → 다음 단계 안내
```
