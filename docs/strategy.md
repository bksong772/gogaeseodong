## 매출 2배 레버 3가지

### 레버 1. 에어비앤비 리스팅 최적화 (즉시 효과)
- 그림의방 소개글 외부채널 문구 제거 → 알고리즘 노출 복구
- 3개 방 모두 제목에서 `#미스터멘션` 제거
- 각 방 고유한 소개글로 교체 (현재 3개 방이 거의 동일한 소개글)
- 그림의방 → 옥탑방 + 드래곤볼/귀멸의칼날 강조로 차별화

**그림의방 확정 제목:**
```
[부산 로컬] 달동네 고택 옥탑방 독채 · 드래곤볼,귀멸의칼날+빔프로젝터 있는 그림의방
```

### 레버 2. 직예약 비중 확대
- 방식: Tally 폼 없이 **카카오 채널 + 계좌이체**로 운영 (수수료 절감)
- 카카오 채널 자동응답에 직예약 5% 할인 안내
- 사이트에서 카카오 채널 링크로 바로 연결

### 레버 3. 패키지 상품
- 서동 하루 패키지: 숙박 + 부엌 + 영화 → 120,000원

## 브랜딩: 서동 유니버스

슬로건: "아직 모르는 부산이 있습니다"
세계관: ARRIVAL → STAY(여관) → EAT(부엌) → WATCH(영화) → REMEMBER(가이드북)

공간별 캐릭터:
- 취향의방: "내 취향이 인테리어가 된 방"
- 그림의방: "옥탑방이 있는 달동네 아지트 · 드래곤볼과 빔프로젝터"
- 필사의방: "쓰는 행위로 치유받는 방"
- 서동부엌: "동네 사람의 공동 부엌"
- 서동영화: "서동 사람들의 작은 극장"

## 마케팅 전략

핵심 전환:
- 기존: 부산 여행 → 에어비앤비 → 고개서동 (우연)
- 목표: 서동 여행 검색 → 고개서동 필수 방문 (의도)

SEO 핵심 키워드: 부산 혼자여행 숙소, 부산 감성숙소, 부산 공유주방, 금정구 숙소

## 서동 여행 가이드북

컨셉: "아직 지도에 없는 서동"
- 형식: **HTML 파일 → Vercel 배포** (Notion 대신, 더 깔끔한 UI)
- 예약 확정 후 카카오 자동응답으로 링크 발송

- 3단계: 실물 인쇄 (2개월) → 체크인 시 제공

## OUR STORY 페이지 리라이팅 전략 (2026-03-18)

서브에이전트: 브랜딩디렉터 + 마케팅전략가

### 핵심 진단
문장 품질 문제 아님. 구조 문제.

**3가지 단절 지점:**
1. 에피소드(방 3개·슈퍼·부부싸움)가 파편처럼 나열 → 묶는 주제문 없음
2. "그래도 돌아왔습니다" 힘 없음 → 떠남→돌아옴 사이 동기(공백) 빠짐
3. 스토리 → HOST CURATION 완전 단절 → "죄송합니다"로 끝난 감성이 서비스 나열로 튐

### 적용한 수정 5가지
1. 에피소드 앞 우산 문장: "그때 저는 다들 이렇게 사는 줄 알았어요"
2. 떠남-돌아옴 사이: 타지에서 계단 소리가 생각났다는 1-2줄 삽입
3. "죄송합니다" → 철학 선언: "지우려고도 하지 않았어요"
4. HOST CURATION 앞 브릿지: "그래서 이걸 하기로 했습니다"
5. CTA 전 상상 유도: "이 동네는 걸어봐야 보입니다"
