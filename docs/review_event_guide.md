필사# 고개서동 블로그 리뷰 이벤트 가이드

> 고객에게 전달하는 공식 가이드. 웹사이트 / 카카오 채널 / 체크인 안내문에 활용.

---

## 혜택

서동부엌 또는 서동영화 **1시간 무료**
- 기본 이용: 3시간 (45,000원)
- 리뷰 작성 시: 1시간 무료 제공 → **15,000원 혜택**
- 추가 시간 이용 시 별도 비용 발생

---

## 리뷰 작성 조건

### 필수 조건
- [ ] **플랫폼**: 네이버 블로그 또는 티스토리
- [ ] **사진**: 공간 사진 **5장 이상** 포함
- [ ] **글자 수**: **500자 이상**
- [ ] **키워드**: "서동여관" 또는 "고개서동" 중 **1개 이상** 본문에 포함
- [ ] **내용**: 숙박한 방 이름 + 실제 이용 후기 포함
- [ ] **발행 기간**: 체크아웃 후 **14일 이내** 발행

### 제출 방법
체크아웃 후 → 카카오 채널 **"고개서동"** 으로 발행된 포스팅 URL 전송

---

## 혜택 지급 방법

포스팅 URL 확인 후 → 카카오 채널로 무료 이용권 코드 or 예약 시 할인 적용 안내

---
